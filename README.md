# L'OBSIDIAN | Fine Dining Concept

A highly polished, cinematic front-end website designed for a luxury Michelin-star style restaurant. The design focuses on moody chiaroscuro lighting, elegant typography, and smooth micro-interactions to deliver a premium user experience.

## 🌟 Features
- **Cinematic Parallax Hero:** Expansive, atmospheric landing section.
- **Interactive Sliding Navigation:** A hidden full-screen drawer menu featuring oversized, hover-triggered image swaps.
- **Asymmetric Editorial Layouts:** Scroll-triggered content blocks with photography that bleeds across the grid, mirroring high-end food magazines.
- **Reservation Modal:** A fully styled booking overlay integrated with Formspree for live data capture.
- **Fully Responsive:** Optimized for both desktop and mobile viewing.

## 🎨 Tech Stack & Design
- **Core:** HTML5, CSS3, Vanilla JavaScript
- **Typography:** Cormorant Garamond (Serif) & Montserrat (Sans-serif)
- **Palette:** Deep Obsidian, Aged Parchment, Gold Accents, Blood Orange
- **Animations:** CSS Transitions & Intersection Observer API for scroll reveals.
